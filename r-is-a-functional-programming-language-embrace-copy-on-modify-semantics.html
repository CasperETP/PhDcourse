2 R is a functional programming language, embrace copy on modify semantics

x <- 3 x <- x + 3 # Right side is evaluated and copied. X is not modified

2.0.1 R has the S4 object system, but this is not a true part of the R semantics.

2.0.2 Functions may input several variables (parameters), may return one variable

my_adder_function <- function(parameter_1, parameter_2) { return (parameter_1+parameter_2) }

2.0.3 All operators are functions

3 + 3 +(3,3)

2.0.4 Note how backticks `` is used for variable names that start with restricted characters - this goes for column names as well
